# Divinest

Divinest is an e-commerce platform for delivering premium ,thoughtfully crafted religious products.

## Project Structure
- frontend/  → User-facing website
- backend/   → API, database, business logic

## Tech Stack
- Frontend: HTML, CSS, JavaScript (React later)
- Backend: Python (FastAPI)
- Database: PostgreSQL
